## About
Understand what an API is, and how to use it to gather data.


## Learning Objectives

*You will be able to:*

- Explain what an application programming interface (API) is.
- Explain the very basics of HTTP and how it's used to get data from the web.
- Use the `requests` library to submit HTTP requests from Python.
- Use a free Python API to access data.
- Evaluate API documentation.

## Prerequisites
- Interpret and use Python dictionaries
- Build `pandas` `DataFrames` from dictionaries
- Perform simple data manipulation on `pandas` objects
- Build `for` and `while` loops in Python
- Use `pip install` for package management
